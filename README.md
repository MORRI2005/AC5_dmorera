AC5_DAVIDMORERA

La AC5 de la asignatura de programación creativa trata del juego de la vida de conway, dentro de este juego no hay jugadores, sino que hay unas celulas las cuales sus vidas dependen de 4 reglas, a partir de estas reglas hay celulalas que nacen y celulas que mueren. Las reglas son: la celula sobrevive con dos o tres vecinas vivas, la celula muere de aislamiento o superpoblación y una celula muerta revive unicamente si tiene exactamente tres vecinas vivas. La idea de este proyecto es recrear este juego a partir de p5.js.

Por mi parte he integrado unas cuantas variaciones el juego, el primero es el hehco de cambiar de color, el segundo es la posibilidad de aumentar la velocidad y la más importante la oportunidad de crear vida. El usuario puede clickar cualquier recuadro en el canvas y crear así una celula, pero también hay la opción de colocar un glider, el cual es un tipo de nave espacial que se puede crear en ocasiones extrañas. También hay disponible otro tipo de naves espaciales como una nave pesada y una nave mediana.
